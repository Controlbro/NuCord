NuCord migration notes

1. Run setup.php once after uploading these files. It uses CREATE TABLE IF NOT EXISTS and checks for each new column before ALTER TABLE, so it updates the schema without deleting existing users, friends, conversations, or messages.
2. Make sure these folders exist and are writable by PHP on shared hosting / DirectAdmin:
   - uploads/avatars/
   - uploads/messages/
3. Place the notification sound at:
   - sounds/notify.mp3
4. Demo seed data is optional from setup.php. Do not seed demo data on a production site unless you want demo accounts.
5. After setup succeeds, delete setup.php or set SETUP_KEY in config.php and only access setup.php with the secret key.
