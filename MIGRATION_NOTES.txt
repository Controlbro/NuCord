NuCord migration notes

1. Upload the files, edit config.php with the existing MySQL credentials, then run setup.php once after upload.
2. setup.php uses CREATE TABLE IF NOT EXISTS and additive ALTER TABLE checks. It does not drop tables or wipe users, messages, friends, groups, or uploads.
3. Make these paths writable/readable by the web server as needed:
   - /uploads/avatars/
   - /uploads/messages/
   - /sounds/
4. Keep the notification sound at /sounds/notify.mp3.
5. Optional site password gate:
   - Set SITE_PASSWORD_ENABLED to true in config.php.
   - Put a password_hash() value in SITE_PASSWORD_HASH.
   - Users will unlock site-password.php before login/register.
6. Optional Google reCAPTCHA:
   - Set RECAPTCHA_ENABLED to true in config.php.
   - Put your Google v2 checkbox site key in RECAPTCHA_SITE_KEY.
   - Put your Google secret key in RECAPTCHA_SECRET_KEY.
7. The app now stores timestamps normally in MySQL and formats message times in the visitor's browser timezone.
